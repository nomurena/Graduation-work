# Graduation-work
## サービス概要
自分磨きの助けになる自分専用の美容記録アプリ。

## このサービスへの思い・作りたい理由
美容の記録をスマホの写真フォルダに保存しても、整理しにくく比較ができない。

SNSでは人に見せる前提なので、自分専用の記録としては使いづらい。

スマホの写真フォルダにはメイク・スキンケア・髪の変化を撮った写真がたくさんあるのに、
過去の写真が埋もれて比較できない。
SNSに載せるほどでもない記録は、結局フォルダで迷子になる。

「“自分だけのカルテ”を気軽に書ける場所があれば良いのに」

そう思ったことが、このアプリを作る決め手になりました。

## ユーザー層について
垢ぬけを楽しみたい・美容を楽しみたい女性
日記をつけたりが好きな方

## サービスの利用イメージ
毎日のメイク・ヘア・スキンケアを写真付きで記録

カレンダーで日付ごとの記録を確認

比較しながら自分の変化を楽しむ

公開ではなく「自分専用ノート」として気軽に書ける
↑個人使用前提
## ユーザーの獲得について
InstagramやX（旧Twitter）などの美容アカウントを活用し、
「自分磨き記録ノート」として発信して周知していく。

## サービスの差別化ポイント・推しポイント
元美容師として、お客様カルテを長年書いていた経験を活かし、
どのように記録すれば次回見返したときに状態が分かりやすいかを熟知。
プロ目線で変化が分かる記録方法に特化

一般的な写真投稿アプリ（既存の写真アルバム,
googleフォト等）とは異なり、プロ視点での美容記録ノートが作れる。

公開前提ではない安心感がある。他人の目を気にせず、肌荒れの日も、失敗メイクの日も、自分のために残せる。SNS（ピンタレスト、lullucos）だと、公開のリスクがあって気軽に失敗を投稿できない。

## 機能候補
機能	内容
投稿CRUD機能	タイトル・内容・画像・日付を登録・編集・削除
カレンダー機能	日別の美容記録を表示（simple_calendar使用）
写真アップロード機能	A
